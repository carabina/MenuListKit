## Requirements

* iOS 9.0+
* Swift 4.0+
* Xcode 9.0+

## Installation

### CocoaPods

MenuListKit is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'MenuListKit'
```
